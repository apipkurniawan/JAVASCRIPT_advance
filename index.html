2. Mengenal Closures pada javascript :

  • Execution Context
    • Creation phase (konsep hoisting) :
      • ketika javascript pertama kali dijalankan(run) maka akan mencari keyword function terlebih dahulu dan akan diset oleh function/code itu sendiri
      • kemudian akan mencari keyword var/variable dan akan diset menjadi undefined
      • javascript mendefiniskan window = global object, dan mendefinisikan this = window
      • function akan membuat Local Execution Context yang didalamnya terdapat creation dan execution phase
    • Execution phase :
      • Mengeksekusi Javascript baris perbaris
      • Untuk melihat bagaimana javascript mengeksekusi code kita : (http://pythontutor.com/javascript.html)
  • Closure
    • Closure adalah sebuah function dikembaikan oleh function yang lain, yang memilik akses ke lingkungan saat ia diciptakan.
    tujuannya adalah untuk membuat function factories.

3. Arrow Function pada javascript :

  • Function terdiri dari 2 yaitu function decralation dan function expression
  • Arrow Function adalah bentuk lain yang lebih ringkas dari function expression
  • Tidak bisa membuat Arrow Function pada constructor function
  • Arrow function tidak memiliki context 'this'

5. Template Literal pada javascript :

  • Template literal hanya bisa dibuat menggunakan tanda back tick (``)
  • Bisa membuat Multi-line string
  • Bisa membuat Embedded Expression
  • Bisa membuat HTML Fragments (yang sering digunakan setiap framework)
  • Bisa melakukan Expression Interpolation
  • Bisa membuat Tagged Template Literals untuk escaping/sanitize HTML Tags, Translation & Internationalization, Styled Components

7. About Looping in Javascript :

  • for..of , forEach, for..in, for()
  • forEach adalah sebuah looping yang hanya bisa mengulang/menelusuri pada array saja, string tidak bisa
  • for..of adalah sebuah looping yang bisa mengulang/menelusuri object-object yang iterable
  • for..in adalah sebuah looping yang hanya bisa mengulang/menelusuri pada enumerable(property pada object) saja
  • Iterable Object :
    1. String
    2. Array
    3. Arguments/NodeList (Arguments adalah Array untuk menampung argument2 pada function) (NodeList adalah ketika melakukan query pada DOM)
    4. TypedArray
    5. Map
    6. Set
    7. User-defined Iterables
